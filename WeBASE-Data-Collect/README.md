## 简介
WeBASE-Data-Collect为数据监管服务数据导出和分析子系统，主要提供以下功能：

- 支持配置多个链，并通过添加区块链前置来获取区块链数据
- 拉取链上数据，包括区块、交易、交易回执
- 支持配置合约和用户信息，解析交易
